# 대한산업AI 랜딩 페이지

대한산업AI 주식회사의 기업 자동화 솔루션 랜딩 페이지입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`으로 확인합니다.

## 구성

- `app/page.tsx`: 랜딩 페이지 본문과 섹션 데이터
- `app/layout.tsx`: 메타데이터와 루트 레이아웃
- `app/globals.css`: 네이비 기반 고급형 디자인, 반응형 레이아웃
- `components/site-header.tsx`: 반응형 헤더와 모바일 메뉴
- `components/hero-canvas.tsx`: 히어로 모션 배경
- `components/use-case-tabs.tsx`: 적용 사례 탭
- `components/contact-form.tsx`: 문의폼 상태 처리

- 실제 솔루션 화면 캡처 또는 데모 영상
