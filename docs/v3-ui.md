# 공통 UI 구성

## 글씨체

- `Lato`, `Noto Sans KR`

## Top bar

### 좌측

- 로고
- 링크: `/v3`
- 의도: v3 메인 페이지로 이동한다.

### 중앙

- Product
- Industries
- Resources
- Company
- 의도: 사이트의 큰 정보 구조를 드롭다운으로 보여준다.

### 우측

- CTA: 문의하기
- 링크: `/v3/contact`
- 의도: 모든 페이지에서 상담 전환을 고정한다.

### 배경

- 기본 상태: 흰색 배경을 사용한다.
- 최상단 상태: 히어로 섹션 위에서는 배경을 투명하게 바꿔 히어로 이미지가 탑바 뒤로 이어져 보이게 한다.
- 스크롤 상태: 사용자가 아래로 이동하면 흰색 배경으로 전환한다.
- 높이와 크기: 최상단과 스크롤 상태 모두 같은 높이와 크기를 유지한다.

### 컴포넌트

- 좌측 로고: 기존 로고 유지.
- 좌측 로고와 업체명: 탑바 배경 상태를 따라 색상이 바뀐다. 최상단 투명 상태에서는 히어로 위에서 보이도록 밝은 색을 사용하고, 스크롤 후 흰 배경에서는 어두운 색으로 전환한다.
- 중앙 메뉴: 검정 글씨, 얇고 깔끔한 인상.
- 우측 CTA: 투명 배경, 글씨와 같은 색의 각진 스트로크.
- 모바일: 메뉴 버튼으로 열고, 드롭다운 항목은 세로로 모두 보이게 한다.

### 드롭다운

- 적용 대상: 중앙 메뉴만.
- 중앙 메뉴 hover 시 테두리가 둥근 네모 패널을 띄운다.
- 상위 메뉴에서 드롭다운 패널로 마우스를 이동하는 동안 패널이 닫히지 않아야 한다.
- 패널 위에 마우스가 올라가 있는 동안에도 드롭다운은 계속 열린 상태를 유지한다.
- 클릭 없이 hover만으로 패널 안 링크를 바로 클릭할 수 있어야 한다.
- 패널은 현재보다 넓게 사용한다.
- 구성:
  - 좌: 대표 이미지 1개, 그룹 설명 1문장.
  - 우: 세부 페이지 제목 목록.
- 대표 이미지는 섹션을 상징하는 이미지로 사용한다.
- 설명은 아주 짧게 쓰고, 페이지별 설명은 화면에 노출하지 않는다.
- 실제 링크가 있는 항목은 페이지로 이동한다.
- 준비중 항목도 준비중 stub 라우트로 이동한다.
- focus 시 별도 강조 색상이나 열린 상태를 강하게 만들지 않는다.
- hover 색상은 과한 면 배경보다 텍스트 색상 변화 중심으로 처리한다.
- 검증: 메뉴 hover, 패널 hover 유지, 패널 링크 클릭, 모바일 터치, 링크 이동.

## 카드 형태

### 기본 카드

- 용도: 반복 항목, 기능 목록, FAQ 보조 정보.
- 구성: 짧은 제목, 1문장 설명, 필요 시 보조 링크.
- 제한: 카드 안에 카드를 넣지 않는다, 뱃지를 달지 않는다.
- 제목 문구 제한: 핵심 기능만 명사형으로 넣는다.
- 설명 문구 제한: 핵심 기능에 대한 설명만 짧고 명확하게 적는다.
- 디자인: 테두리 끝이 살짝 둥근 형태.
- 동작: hover 시 살짝 커지는 동적 구성.

### 이미지 카드

- 용도: 이미지가 들어가는 카드.
- 구성: 이미지가 가득 찬 카드.
- 디자인: 테두리 끝이 살짝 둥근 형태.
- 동작: hover 시 살짝 커지는 동적 구성.
