# v3 UI 점검 체크리스트

마키나락스 레퍼런스를 그대로 복제하기보다, 대한산업AI의 신뢰감과 산업 자동화 메시지가 먼저 보이도록 확인한 항목입니다.

## 확인한 문제

- [x] 상단 문의 CTA가 일부 페이지에서 파란 사각 버튼으로 보여 레퍼런스의 흰 탑바 / 검정 outline pill 톤과 어긋남.
- [x] 제품 쇼케이스 섹션의 내부 라벨 여백이 과해 중간 영역이 비어 보임.
- [x] 홈 히어로는 큰 산업 비주얼과 핵심 문구가 잘 보이나, CTA 톤은 탑바와 더 일관되게 정리 필요.
- [x] 모바일 홈에서 제목 줄바꿈은 안정적이나 버튼과 진행바가 너무 빽빽해질 수 있어 유지 가능한 최소 간격 확인 필요.
- [x] 서브페이지 히어로는 정보 구조는 명확하나, 탑바 CTA 스타일 우선순위가 페이지별로 달라 보임.

## 반영 방향

- [x] 탑바는 흰 배경, 검정 로고, 중앙 메뉴, 우측 outline pill CTA로 고정.
- [x] 홈은 큰 산업 현장 비주얼과 핵심 문구 중심으로 유지.
- [x] 제품 쇼케이스는 빈 여백이 아니라 제품 설명과 미디어가 같은 화면에서 읽히도록 조정.
- [x] 각 페이지는 메인 / 제품 / 회사소개 / MVP / 문의 목적이 첫 화면에서 바로 구분되게 유지.
- [x] 빌드와 주요 화면 캡처로 최종 확인.
